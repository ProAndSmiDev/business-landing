### Краткая инструкция по работе

#### 1. Работа с ветками
      - master: вся работа над сайтом происходит на этой ветке

#### 2. Кодстайл:
      1. Все размеры указываются в em, делается через миксин em()
      2. Все цвета задаются строго в hsl формате
      3. JS-скрипты пишутся на ES2015+ (исключения составляют сторонние плагины и библиотеки)
      4. Разметку пишем строго по BEM в 1-2 класса (класс блока и класс элемента)
      5. Блокам стили задаются отдельно, без наследования
      6. Элементам стили задаются через наследование
      7. Не использовать сокращенные свойства и изменять только то, что необходимо

      P.S. Все свойства и стили - не влияющие на содержимое блока являются свойствами элемента (bock__name), остальные - свойствами блока (block-name)

#### 3. SVG:
      - в папку app/assets/svg кидать только свг иконки с англ названиями, например icon-close. Сборщик сгенерирует спрайт из всех иконок с id, указанными в имени файлов. Конечный путь до svg-спрайта - docs/img/sprite.svg.
